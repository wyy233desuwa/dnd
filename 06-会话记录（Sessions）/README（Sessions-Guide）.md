# 会话记录说明（Sessions Guide）

每次会话使用独立目录：

```text
会话-001（Session-001）/
├── 摘要（Summary）.md
├── 事件日志（Events）.md
├── 关键骰点（Key-Rolls）.md
└── 战斗归档（Combat-Archive）.md
```

- `事件日志`：游戏过程中持续追加，作为历史审计记录。
- `摘要`：暂停或结束时更新，便于人类回顾。
- `关键骰点`：只记录会影响长期状态或剧情的骰点。
- `战斗归档`：保存本会话已经结束的战斗状态。

历史日志不应成为当前 HP、位置或任务状态的权威来源。
