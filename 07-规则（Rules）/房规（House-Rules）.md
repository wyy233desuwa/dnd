# 房规（House Rules）

> 本战役以 D&D 2024 Revised Fifth Edition / SRD 5.2.1 为规则基线。  
> 本文件只记录明确偏离、补充或固定解释的内容。

## 经验与升级

- 升级方式：里程碑。
- 里程碑通常来自：
  - 完成一个重要任务；
  - 解决主线阶段的核心问题；
  - 角色经历显著成长；
  - 发现并处理足以改变局势的重要真相。
- 升级通常在安全地点完成。
- 升级可以在长休后或会话结束时生效。
- 升级前必须先更新会话摘要和当前存档。
- 职业、专长、法术、属性及其他升级选择必须由玩家确认。
- Agent 可以解释选项，但不得替玩家自动选择。
- 不因为单纯重复战斗或刷取普通敌人自动获得等级。

## 难度与公平

- 默认战斗难度：标准。
- 重要战斗可以达到困难，但必须通过环境、情报或明显迹象预告风险。
- 高致命遭遇只能在玩家主动进入明确危险区域，或剧情已经充分预告时使用。
- Agent 不暗改骰点。
- Agent 可以在战斗开始前依据合理信息调整遭遇设计，但战斗开始后不得无依据改变敌人数据。
- 敌人会根据自身知识行动，不会自动知道玩家角色的全部能力。
- 低智力生物不会使用超出其认知的完美战术。
- 智慧敌人可以集火、设伏或撤退，但必须符合其目标和已知信息。

## 信息公开

- 敌人准确 HP：默认不公开。
- 敌人状态描述：
  - 健康：尚未受到明显伤害；
  - 受伤：已经受到可见伤害；
  - 重伤：伤势明显，行动开始受到影响；
  - 濒死：接近失去战斗能力。
- 检定 DC：默认不提前公开准确数字。
- 当角色能够合理判断任务难度时，可以说明"容易、一般、困难、极难"。
- 判定结束后，可以在玩家询问时公开普通检定 DC。
- 隐藏检定的具体结果不在当场公开，但必须记录在会话事件或关键骰点中。
- 玩家已经通过可靠方式确认的事实，不得继续用隐藏检定反复否定。

## 死亡与复活

- 死亡豁免采用 D&D 2024 默认规则。
- Agent 不得为了保护角色而修改死亡豁免骰点。
- 敌人是否攻击倒地角色取决于其目标、智力、性格和战斗局势。
- 角色死亡是实际风险，但不应由毫无预警、无意义且不可避免的事件造成。
- 角色死亡后暂停推进，由玩家决定是否：
  - 尝试规则允许的复活；
  - 创建新角色；
  - 让原角色以其他合理方式继续参与；
  - 结束或调整当前战役。
- 复活必须同时满足：
  - 有对应规则能力；
  - 世界中存在能够执行的施法者或物品；
  - 支付相应资源和代价；
  - 灵魂愿意且能够返回。
- 不使用没有铺垫的神迹无条件取消死亡结果。

## 临时裁定流程

规则无法立即确定时：

1. Agent 明确说明存在规则疑问；
2. 给出一个不明显偏袒任何一方的临时裁定；
3. 继续当前场景，避免长时间中断；
4. 将裁定写入"待确认裁定"；
5. 会话结束后由玩家决定是否成为永久房规。
