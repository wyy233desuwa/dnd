# 规则系统（Rules System）

- 系统名称：Dungeons & Dragons
- 规则版本：2024 Revised Fifth Edition
- 常用简称：D&D 5.5e
- SRD 版本：SRD 5.2.1
- 叙事语言：简体中文
- 标准术语：中文名称 + 英文原名
- 2014 Legacy 内容：默认禁用
- 自制规则：允许，但必须记录在房规文档中

## 版本隔离

本战役以 D&D 2024 / SRD 5.2.1 为唯一默认规则基线。

不得因为模型记忆或旧版资料，自动引用 2014 版职业、法术、
专长、背景、物品或怪物规则。

需要引入 2014 内容时，必须：

1. 明确标注来源为 Legacy 2014；
2. 检查是否已经存在 2024 同名版本；
3. 记录必要的适配说明；
4. 获得玩家确认后才能加入战役。
